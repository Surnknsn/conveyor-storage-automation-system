# Conveyor Storage Automation System

## Overview
This project is an automated conveyor-to-storage system designed to transfer materials into selected storage positions. The system is developed using both PLC and Arduino platforms to demonstrate industrial and microcontroller-based control systems.

## System Versions
### PLC-Based System
- Industrial control system using PLC
- Suitable for real factory applications
- Stable and reliable operation

### Arduino-Based System
- Microcontroller-based control using Arduino Mega
- Relay module for actuator and conveyor control
- LCD display for system status

## System Components
- Conveyor Belt
- Storage (3 positions)
- Linear Actuator
- Control Panel (Switches & Indicators)
- Sensors

## Features
- Automatic material transfer
- Storage position selection
- Conveyor and actuator control
- Emergency stop system

## Working Principle
1. Start system
2. Select storage position
3. Actuator moves to target
4. Conveyor transfers object
5. Sensor detects object and stops system
6. System returns to initial position

## Technologies Used
- PLC (Mitsubishi)
- Arduino Mega
- Relay Module
- Electrical Wiring Design

- /plc-version → PLC system
- /arduino-version → Arduino system
